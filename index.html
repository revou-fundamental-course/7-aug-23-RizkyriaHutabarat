Luas Segitiga

Rumus luas segitiga yaitu:

L = (A X T) / 2

Dimana:

L = Luas

A = Alas

T = Tinggi

Keliling Segitiga

Rumus keliling segitiga yaitu:

K = A + B + C

Dimana:

A = Sisi A

B = Sisi B

C = Sisi C
